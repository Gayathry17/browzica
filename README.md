# Getting Started
### Clone the repo


```
git clone https://github.com/Gayathry17/browzica.git
cd browzica
```
### Install packages
```
npm install
```
#### or
```
yarn install
```
### Start the development server
```
npm start
```
#### or
```
yarn start

```
